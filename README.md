# 🚀 PCB for Nova-X Deauther (BW16 Version)

<table border="0" cellpadding="0" cellspacing="0">
  <tr>
    <td valign="top" style="border: none; background: transparent; padding-right: 20px;">
      An open-source, custom PCB specifically designed to run the <strong>Nova-X deauther firmware</strong>. This board is tailored for the <strong>BW16 (RTL8720DN)</strong> module, bringing dual-band Wi-Fi network testing into a clean, dedicated hardware layout.
      <br><br>
      This repository provides all the necessary Gerber files for free. Download them, order your boards, and build your own hardware!
      <br><br>
      <h3>📦 Required Components (BOM)</h3>
      To fully assemble this board, you will need:
      <ul>
        <li><strong>1x</strong> BW16 (RTL8720DN) Module</li>
        <li><strong>1x</strong> 0.96" OLED Screen (I2C)</li>
        <li><strong>4x</strong> Through-hole 6x6mm Tactile Buttons</li>
        <li><strong>1x</strong> SMA Edge Connector</li>
        <li><strong>1x</strong> IPX Connector (for the board)</li>
        <li><strong>1x</strong> IPX to IPX Cable</li>
        <li><strong>1x</strong> Wi-Fi Antenna (Dual-Band recommended)</li>
      </ul>
      <br>
      <h3>✨ Features</h3>
      <ul>
        <li><strong>Made for Nova-X:</strong> Fully compatible layout designed specifically around the Nova-X firmware.</li>
        <li><strong>Dual-Band Hardware:</strong> Utilizes the BW16 module for 2.4GHz and 5GHz network analysis and deauth testing.</li>
        <li><strong>Open & Free:</strong> All Gerber files are open-source and ready to send straight to manufacturers like JLCPCB or PCBWay.</li>
      </ul>
